3D Modelling Software

3D modelling is the process of creating three-dimensional digital representations of objects or scenes. 3D modelling software is crucial for the design and development of the 3D gorilla prototype. 3D modelling software enables users to visualise designs and concepts in three dimensions, allowing uses to create accurate models with exact measurements and dimensions. Furthermore, 3D modelling software enables cost-effective prototyping, allowing users to transform ideas to produce a functional prototype or product. Some examples of 3D modelling software tools include Blender, Sketch-up, and Fusion 360.

Blender, created by Ton Roosendaal and launched in January 1995, is a 3D modelling software widely used by independent artists and animators (ref). Blender has been chosen as the 3D modelling software that will be used to design and reconstruct the 3D gorilla once it has been 3D scanned. Blender was chosen for use in the project as it is entirely open source and supports a wide range of tools for modelling and sculpting, which will aid in producing an accurate design of the 3D gorilla.
